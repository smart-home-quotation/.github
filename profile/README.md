## 智能家居报价系统

*  控制台  
控制台是登录后的业务首页，用来快速查看系统统计和进入高频操作。

*  客户管理  
客户管理负责沉淀客户基础资料，供图纸和报价后续复用。

*  报价管理  
报价管理用于创建和维护人工报价单，并输出正式版本。

*  图纸管理  
图纸管理负责图纸文件台账、预览状态跟踪和后续编辑入口。

*  产品管理  
产品管理是报价和库存流程依赖的商品主数据入口。

*  库存管理  
库存管理用于跟踪商品当前数量、预警状态和出入库流水。


*  采购管理  
采购管理用于维护采购单头、商品明细和收货动作。

*  品牌与分类  
品牌与分类用于维护产品主数据依赖的字典信息。

*  用户管理  
用户管理用于维护系统登录账号和角色权限。
